# ScanHexDump

16進ダンプのテキストファイルをバイナリファイルに変換するプログラム。

## アルゴリズム
- 大文字/小文字は無視する。
- 行頭の1カラムをチェックする。
  - `Add` この行は無視する。
  - `Sum` 縦方向のチェックサム行として認識、チェックを行う。
- 空行はスキップする。
